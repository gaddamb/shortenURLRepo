Steps to run the application:
Application has been built using SpringBoot, Angular JS, Maven and Java.
1. Prerequisites: Install JDK 1.8, Maven-3.6.0
2. Provide the bin directory path of Java & Maven to 'PATH' environment variable. Please follow the steps to set the Path variable: https://maven.apache.org/install.html
3. Open command prompt and run the below command (Assuming you have unzipped the folder in C: drive): c:\shortenURLRepo-master>mvn spring-boot:run
4. If no other application running in your local installed server, then you can view the application by entering the url at browser: http://localhost:8080/
